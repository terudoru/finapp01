# 📈 高度株価分析＆AI予測ダッシュボード (QUANT VIEW)

先進的な機械学習（XGBoost）とプロフェッショナル仕様のテクニカル・ファンダメンタル指標を組み合わせた、ローカル完結型の強力な投資分析ツール（ダッシュボード）です。

システムの心臓部である `quant_engine.py` と、美しくインタラクティブな `streamlit_app.py` によって柔軟な分析と高速なレスポンスを実現しています。APIによる外部LLM（Gemini等）への依存はなく、完全にプライベート環境で高度な計算を実行します。

## ✨ 主な機能

### 1. **AI予測と「確率」の表示 (XGBoost)**
単なる「上がる・下がる」だけでなく、「上昇確率 65%」のようにAIの自信度を表示します。最新の相場動向（テクニカル・カレンダー・出来高など）から、網羅的なハイパーパラメータ最適化（Auto-Tune）を経て高精度な予測を弾き出します。

### 2. **高度テクニカルチャート ＆ マルチペイン表示**
インタラクティブなPlotlyチャートを用い、ローソク足に加えて「ボリンジャーバンド」「RSI」「MACD」などのプロ向け指標をマルチペインで同時に表示します。

### 3. **バックテスト シミュレーション**
過去から現在までの市場データを使用し、「AIのシグナル通りに取引していた場合」の資産曲線を計算します。単純に持ち続けた場合（Buy & Hold戦略）とのパフォーマンス格差（勝率、ドローダウン等）を一目で比較できます。

### 4. **マクロ指標感応度 ＆ ファンダメンタルズ分析**
対象銘柄が「市場全体 (S&P 500)」「VIX (恐怖指数)」「原油・金」などに対してどの程度連動するかを分析（ベータ値）する他、PERや自己資本比率等の重要な企業基礎情報もダッシュボード上で完結して確認できます。

### 5. **ポートフォリオ相関分析 ＆ イベントカレンダー**
登録した保有銘柄同士が「実は同じ動きをする（分散できていない）」リスクをヒートマップで可視化します。また、決算発表日や配当落ち日のスケジュール表とCSVダウンロード機能も備えています。

### 6. **ニュース感情分析 (FinBERT連携)**
AIベースの自然言語処理を活用し、直近のニュース見出しからマーケットの「センチメント（楽観的/悲観的）」を数値化して提示します。

## 🛠 使い方（ローカル環境）

### 1. 必要な環境のセットアップ
ターミナルを開き、以下のコマンドで必要なライブラリをインストールしてください。

```bash
# Python仮想環境のアクティベート（推奨）
source .venv/bin/activate

# 依存モジュールのインストール
pip install -r requirements.txt
```

### 2. アプリの起動

**A. WEBブラウザでGUIダッシュボードを使う（推奨）**
ターミナルで以下を実行します。
```bash
streamlit run streamlit_app.py
```
実行後、自動的にブラウザが立ち上がり `http://localhost:8501` でアプリを利用できます。

**B. コマンドライン（CUI）で速報レポートを確認する**
```bash
python predict.py
```
ターミナル上でXGBoostの最適化学習が高負荷で実行され、本日の予測結果、バックテスト精度、SHAP（判断根拠）、利確・損切りラインがテキストベースで超高速出力されます。

## 🚢 将来のVPSデプロイ向け
本リポジトリには `Dockerfile` と `docker-compose.yml` が同梱されています。
デプロイ環境で以下のコマンドを打つだけで、即座にダッシュボードを公開・起動できます。

```bash
docker compose up -d --build
```
